整体架构设计

纯前端单页应用，数据从 jsDelivr CDN 加载

技术选型

纯 HTML/CSS/JS（无框架）

零依赖、零构建、体积小、加载快。GitHub Pages 直接托管，无需编译。

页面结构

┌─────────────────────────────────┐
│  Header: 标题 + 时区选择器  │
├─────────────────────────────────┤
│  [ 赛程 ] [ 积分榜 ] [ 淘汰赛 ]  │
├─────────────────────────────────┤
│              内容区              │
│  • 日期分组 / 小组筛选     │
│  • 比赛卡片列表           │
│  • 积分榜表格             │
│  • 淘汰赛对阵树           │
├─────────────────────────────────┤
│  Footer: 数据来源 / 开源链接  │
└─────────────────────────────────┘

数据流

1. 页面加载 → fetch CDN worldcup.json
2. 解析 JSON → 按 round/group/date 分类
3. 计算积分榜：从比赛结果累加胜/平/负/积分/净胜球
4. 构建淘汰赛对阵树：匹配 round 引用（W89→Match 89 胜者）
5. 时区转换：解析 UTC±N → 转换为用户所选时区
6. 渲染到对应 Tab 内容区
