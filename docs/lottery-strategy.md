# 体彩购买策略

> **目标**：基于 53 场已完赛数据（6/11–7/2，覆盖小组赛 + R32）的盈亏分析，制定可执行的购买策略，跑赢体彩 ~27% 抽水。
>
> **数据基础**：见 [`lottery-pnl.md`](./lottery-pnl.md)，ROI 计算 = (返还 − 投入) / 投入。
>
> **更新日期**：2026-07-10｜**适用阶段**：SF（7/11–7/12）+ 决赛

---

## 一句话策略

**只买 HAD（胜平负）单关或 2 串 1，选赔率 1.30–1.60 区间的强队，每注 ¥2，2–3 倍。**

其他玩法默认不碰。

---

## 五条黄金法则

1. **只买 HAD**，且最低赔率在 **1.30–1.60** 黄金区间 → 历史 ROI **+31.5%**，命中率 **91.7%**
2. **2 串 1 是上限**：同日两场黄金区间，¥2/注，赔率乘积约 2 倍 → ROI **+48.3%**
3. **绝对热门（≤1.30）反直觉地亏**：赔率 1.22 < 保本线 1.29，**不买**
4. **HHAD 只买 ±2 中让球**：强队让球后还能赢才买；**±1 弱让球是陷阱**（ROI −31.4%）
5. **永不买**：TTG 总进球、3 串 1 及以上串关

## 串关策略

### 推荐方案：2 串 1

- **选场**：同日两场，**都是 HAD 黄金区间（1.30–1.60）**
- **注金**：¥2/注，2–3 倍
- **赔率乘积**：约 1.46 × 1.56 = 2.28
- **全中率**：历史 ~75%
- **期望 ROI**：+48.3%

### 选场优化（待验证）

- 优先选**同阶段**（都是 SF 或都是 QF），避免小组赛末轮轮换
- 两场都选**1.30–1.60 中段**（如 1.40–1.55），避开区间边缘
- **不要跨日**串关——同日比赛相关性更强

### 永不触碰

- 3 串 1：全中率从 75% 骤降到 27%
- 4 串 1：6 注全中 0 次，归零
- 5 串 1+：数学上必输

## 资金管理

| 项 | 建议 |
|---|---|
| 单注金额 | 固定 ¥2/注 |
| 倍数 | 2–3 倍，不追输 |
| 单场投入上限 | ¥6（HAD + HHAD ±2 + 倍数） |
| 单日投入上限 | ¥20（最多 3 场 × 2 串 1） |
| 止损 | 单日亏 ¥30 停手 |
| 止盈 | 单日赚 ¥50 收手 |

**不要做的事**：
- ❌ 倍投追输（输了一注下一注翻倍）
- ❌ 全押单场
- ❌ 临时加注（赛前突然觉得"稳了"加倍）
- ❌ 买冷门搏大奖（>3.00 的赔率长期必输）

## 决策流程（每次购买前过一遍）

1. **看 HAD 最低赔率是否在 1.30–1.60**？
   - 否 → **不买**，结束
   - 是 → 进入第 2 步
2. **当天有几场符合条件**？
   - 1 场 → 单买 ¥2 × 2–3 倍
   - ≥2 场 → 2 串 1 ¥2 × 2 倍 + 单买主推场 ¥2 × 1 倍
3. **总投入是否超过单日上限 ¥20**？
   - 是 → 减倍数
   - 否 → 执行
4. **有没有触发反例**？
   - 有 → 删掉该场
   - 无 → 下注
