# 体彩世界页签 — 设计文档

**日期:** 2026-06-17
**状态:** 已实现

## 概述

在现有"历届冠军"页签后新增"体彩世界"页签，展示中国体彩官方世界杯赔率数据，支持模拟投注计算。

## 功能范围

- 展示 `data/lottery-odds.json` 中的赔率数据（12场左右，覆盖1-3天）
- 四种玩法切换：胜平负(HAD)、让球胜平负(HHAD)、总进球(TTG)、半全场(HAFU)
- 全部玩法完整赔率展示（TTG 8选项 0~7+球，HAFU 9选项 胜胜~负负）
- 多选投注：同场同玩法多选项，同场跨玩法多选，不同场自由组合
- 过关投注：自动按 (match, pool) 分组计算串关，不同玩法=不同注
- 注数 = 每场可选数的乘积，每注 ¥2
- 预计奖金 = 倍数 × 2 × 赔率（多个串关时显示范围 min~max）
- 投注金额 = 注数 × ¥2 × 倍数
- 单关/串关标注：● 可单关 / ◐ 须串关
- 与 worldcup.json 赛程匹配，显示国旗和小组信息
- 融入现有深绿+金色主题

## 数据流

```
data/lottery-odds.json (fetch) → lottery.js 渲染
                                      ↓
TEAM_ZH 反向映射 (中文→英文) → 匹配 worldcup.json matches (日期+两队名)
                                      ↓
                              显示国旗 + 小组 + 赔率卡片
```

## 投注计算

- 1倍 = 2元（体彩标准）
- 分组：`getSelectionGroupsByPool()` 按 (matchNum, pool) 分组
- 注数：`computeOddsRange()` 生成所有合法串关组合，每场只取一个池一个选项
- 收益范围：所有串关 odds 乘积的 min~max
- 纯前端模拟，刷新即清空

## 约束

- 无框架、无构建、无 npm 依赖
